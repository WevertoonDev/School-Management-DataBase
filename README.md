# 🎓 School Management Database

Projeto de banco de dados desenvolvido em SQL utilizando SQLite.

## 📌 Funcionalidades
- Cadastro de alunos
- Cadastro de professores
- Disciplinas e turmas

## 🛠️ Tecnologias
- SQL
- SQLite

## 📂 Estrutura
- alunos
- professores
- disciplinas
- turmas

## 🚀 Objetivo
Praticar modelagem e criação de banco de dados relacional.
